# 📚 O QUE VOCÊ PRECISA SABER PARA CRIAR ESTE BOT

## 1️⃣ PYTHON BÁSICO
- **Variáveis**: int, str, float, bool
- **Estruturas**: if/elif/else, for, while
- **Funções**: def, parâmetros, return
- **Orientação a Objetos**: classes, métodos
- **Módulos**: import, from

## 2️⃣ BANCO DE DADOS (SQLAlchemy)
- **Modelos**: criar tabelas com Column()
- **Sessões**: sessionmaker, session.add(), session.commit()
- **Queries**: filter_by(), first(), all()
- **Relacionamentos**: ForeignKey, relationship()

## 3️⃣ API TELEGRAM (python-telegram-bot)
- **Application**: Application.builder().token()
- **Handlers**: CommandHandler, MessageHandler
- **Updates**: processar mensagens e comandos
- **Context**: context.user_data para estado

## 4️⃣ DEPENDÊNCIAS PRINCIPAIS
- **python-telegram-bot**: para o bot
- **sqlalchemy**: para banco de dados
- **python-dotenv**: para variáveis de ambiente
- **cryptography**: para criptografia

## 5️⃣ ARQUITETURA DO PROJETO
- **main.py**: ponto de entrada
- **src/config/**: configurações
- **src/models/**: banco de dados
- **src/commands/**: handlers do Telegram
- **src/services/**: lógica de negócio
- **src/utils/**: utilitários

## 6️⃣ CONCEITOS-CHAVE
- **Async/Await**: programação assíncrona
- **Environment Variables**: .env para senhas
- **Virtual Environment**: isolamento de dependências
- **Git**: controle de versão

## 7️⃣ FUNCIONALIDADES DO BOT
- **Registro de transações**: receita/despesa
- **Categorização automática**: IA simples
- **Relatórios**: mensal, semanal, completo
- **Saldo**: cálculo automático
- **Reset**: zerar dados
